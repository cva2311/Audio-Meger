# AudioMerge.py

AudioMerge is a Python utility for merging mutiple audios from a path into a single audio  

## Installation

Install SOX and python 3.4 and above


## Usage

AudioMerge.py "Path where the audio files to be merged"
 *************** Ex : AudioMerge.py /home/Docs/AudioMerger/wavfiles/ **************************

Make sure the AudioMerge.sh to be present in the same path where the input wavfiles are for merging.


Input Files must have the same # channels
